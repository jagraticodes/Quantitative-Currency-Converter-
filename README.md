# 📊 Quantitative Currency Converter Dashboard

Welcome! This is a high-performance desktop application built as part of the **Pinnacle Labs Developer Internship Program (2026)**. Moving beyond standard conversion math, this project features an analytics-driven financial dashboard experience.

---

## 🚀 Key Engineering Features

- **🌐 Real-Time Global Market Sync:** Automatically connects natively to external API endpoints to capture live-market foreign exchange rates.
- **📉 Dynamic Trend Visualizations:** Features an anti-aliased 7-Day Market Spot Volatility index line graph that dynamically updates on every single conversion.
- **🛡️ Bulletproof Fault Tolerance:** Engineered with smart input-error validations and an automatic offline fallback system to local baseline databases in case of connection dropouts.
- **🎨 Premium Dark Theme UI:** Implemented with a polished, lightweight dashboard framework using pure Java Swing components.

---

## 🛠️ Technology Stack & Architecture

- **Language:** Java Core (JDK 17/21)
- **Framework:** Java Swing & Java 2D Graphics Rendering Engine (`Graphics2D`)
- **Networking:** Native Java HTTP URL Connection Client.

---

## 🖥️ How to Run Locally

1. Open the project folder inside **Visual Studio Code**.
2. Ensure the **Extension Pack for Java** (by Microsoft) is installed.
3. Open `CurrencyConverter.java` and click on **Run**.
